## DVD Bounce Twitch Overlay

A tiny, chaotic little overlay that recreates the classic DVD logo bounce, but for Twitch.

The logo floats around the screen and (eventually 👀) hits a corner.
When it does, it plays a sound so chat can celebrate like it's 2006.

### Perfect for:

- “Guess when it hits the corner” games
- Just Chatting chaos
- Nostalgia-core streams

### Features

- Smooth, GPU-accelerated bouncing
- Corner detection
- Sound effect on corner hit
- Works as an OBS Browser Source
- Lightweight and looped forever

### How to Use

- Put your image file in the folder
- Replace avatar.png in dvd.html
- Add corner.mp3 (or your own sound)
- In OBS → Add Browser Source
- Check Local File
- Set width/height to your stream resolution (e.g. 1920×1080)
- Done. Let it bounce.

### Stream Idea

Tell chat to guess the exact timestamp when it’ll hit a corner.
Winner gets VIP. Or eternal bragging rights.
